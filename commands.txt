any: Allows you to use any mysql command.
setcolor: Retorna a quantidade de cartas de cada cor em um set especifico
bestkeywordcards: Retorna uma lista ordenada do número de keywords de cada carta em que a stormScale tem uma média baixa. Isso significa que as cartas nessa lista tem mais chance de serem imprimidas novamente.
walkerbyrace: Retorna as cartas de todos os planeswalkers de uma certa raça.
coloredlands: Retorna as cartas de terreno que não são básicos e produzem cor (não é colorless)
isallowed: Retorna todos os formatos em que uma carta é permitida
rares: Retorna todas as criaturas de uma certa cor que são raras.
multicolored: Retorna todas as criaturas multicoloridas.
commonkeywords: Para uma cor especifica, retorna uma lista de palavras-chaves que aparecem em cartas daquela cor, ordenada decrescentemente.
creaturetype: Retorna os tipos de uma criatura especifica, junto com a classificação de cada tipo.
ramplands: Retorna todos os terrenos que produzem mais de um de mana por vez.
